This app has its own API for getting statistics about searches. Main API endpoints are:

  • all authors, sorted by the most searched:
    http://127.0.0.1:8000/api/authors/

  • first x most searched authors:
    http://127.0.0.1:8000/api/authors/?limit=x/
    for example http://127.0.0.1:8000/api/authors/?limit=3/

  • specific author by name and surname:
    http://127.0.0.1:8000/api/author/Name%20Surname/
    for example http://127.0.0.1:8000/api/author/Paulo%20Coelho/
