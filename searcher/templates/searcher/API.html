This app has its own API for getting statistics about searches. Main API endpoints are:

  • all found authors, sorted by the most searched:
    http://.../api/authors/

  • first x most searched authors:
    http://.../api/authors/?limit=x

    for example http://.../api/authors/?limit=3

  • specific author by name and surname:
    http://.../api/author/Name%20Surname/

    for example http://.../api/author/Paulo%20Coelho/
